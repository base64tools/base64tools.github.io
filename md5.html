MD5 (Message-Digest Algorithm 5) is a widely used cryptographic hash function that produces a 128-bit hash value. It's commonly used for checksums, file integrity verification, and password storage (though not recommended for security-critical applications due to vulnerabilities).

Example

Input: hello → Output: 5d41402abc4b2a76b9719d911017c592

FAQ

Is MD5 secure for passwords? No, it's vulnerable to collisions and rainbow table attacks. Use stronger hashes like bcrypt or SHA-256.
